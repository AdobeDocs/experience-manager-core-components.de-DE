# Dokumentation zu Adobe Experience Manager-Kernkomponenten

## Dokumentation zu Kernkomponenten

Dies ist das Repository für Adobe Experience Manager-Kernkomponenten ***Dokumentation***. Diese Dokumentation behandelt die Installation, die Verwendung der Authoring-Funktion und die Anpassung der Kernkomponenten.

Ihre Beiträge zur Dokumentation sind willkommen. Siehe die folgenden Ressourcen:

* [contributor.md](contributing.md) Einen Überblick darüber, wie Sie zur Dokumentation beitragen können.
* [guidelines.md](guidelines.md) Einen Überblick darüber, was an Beiträgen erwartet wird und wie Sie Ihre Dokumentationsbeiträge zusammenstellen.
* [code-of-conduct.md](code-of-conduct.md) Einen Überblick über die Verhaltensnormen, die wir von Ihnen erwarten, wenn Sie zu diesem Dokumentationsprojekt beitragen.

## Kernkomponenten-Repository

Die [Adobe Experience Manager-](https://github.com/adobe/aem-core-wcm-components) selbst werden als Projekt in einem separaten Repository gepflegt. Bitte schauen Sie sich dieses Projekt an, wenn Sie technische/Entwicklerdokumentation zu den Komponenten benötigen, zu den Komponenten selbst beitragen möchten oder ein Problem mit den Funktionen oder Funktionen der Kernkomponenten einreichen müssen.
